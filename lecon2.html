🧠 Synthèse

🔁 récursivitė : une fonction s'appelle elle-même (cas de base + réduction). Induction : preuve par récurrence — initialisation + hérédité. Ces deux concepts sont les deux faces d'une même pièce : la récursivité exécute, l'induction prouve.

💼 Application
📂 Je dois explorer une arborescence de fichiers dans un outil de synchronisation. Un parcours récursif (DFS) est bien plus simple à écrire et maintenir qu'une version itérative avec une pile manuelle. J'ai aussi utilisé l'induction pour vérifier que mon parcours visite bien tous les fichiers.
